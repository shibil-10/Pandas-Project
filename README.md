# Indian Engineering Students Placement Analysis

## Project Overview

This project performs an Exploratory Data Analysis (EDA) on a dataset containing information about engineering students in India. The objective is to uncover factors that influence academic performance, coding skills, stress levels, and placement outcomes.

The analysis includes:

- Data Cleaning
- Outlier Detection & Treatment
- Feature Engineering
- Statistical Analysis
- Data Visualization
- Insights & Recommendations

---

## Dataset Information

- Records: 5,000 Students
- Features: 23 Columns
- Domain: Education & Placements
- Source: Kaggle

The dataset contains information related to:

- CGPA
- Study Hours
- Attendance
- Coding Skills
- Internships
- Certifications
- Projects Completed
- Family Income
- Stress Levels
- Sleep Hours
- City Tier
- Placement Status

---

## Project Objectives

This analysis aims to answer the following questions:

1. What is the distribution of CGPA among engineering students?
2. How does CGPA vary across engineering branches?
3. Does studying more hours improve CGPA?
4. How does attendance affect academic performance?
5. What is the stress level distribution among students?
6. Do projects improve coding skills?
7. How do internships influence CGPA?
8. Does family income affect stress levels?
9. What is the gender distribution of students?
10. Do certifications improve coding skills?
11. How do backlogs impact CGPA?
12. Which factors are strongly correlated?
13. How do sleep patterns affect stress and CGPA?
14. Does internet access improve coding skills?
15. Are there differences between city tiers?

## Key Insights

### Academic Performance
- Higher study hours generally correspond to better CGPA.
- Students with higher attendance tend to achieve stronger academic results.
- Backlogs negatively impact overall CGPA.

### Skill Development
- Students completing more projects often exhibit stronger coding skills.
- Certifications contribute positively to coding proficiency.
- Internet access appears to support skill development.

### Student Well-being
- Sleep patterns influence both stress levels and academic performance.
- Stress varies across different income groups.

### Career Readiness
- Internships and projects significantly enhance practical exposure.
- Students with stronger technical profiles are more placement-ready.
